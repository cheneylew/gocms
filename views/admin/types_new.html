创建新内容模型

New Content Type
  • 输入此类型内容的名称。
Options
  • 如果选中，每个内容项将具有以下字段：“标题”，“URL路径”和“主题”。这些是允许Hero将此内容显示为单个网页，包括在博客/主题列表等中的标准项目。
  • 如果选中，您将能够指定有权查看此内容（或将其设为公开）的成员组。
Design
  • 主题目录中的此模板将用于显示此类型的内容。 （如果不适用，请忽略此字段。）
  • （可选）如果设置了此值，则您为每个内容创建网址的“网址路径”框将预先加载一个文件夹，该文件夹将统一此类型的所有内容。例如，将此设置为“articles /”将有助于您为此类型的内容（如“articles / my_article”，“articles / my_other_article”等）创建所有网址。
